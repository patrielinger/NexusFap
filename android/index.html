    Instalacion de Juegos

    Para descargar un juego o app en otra plataforma de que no sea Google play se descargan archivos APK los cuales
    para poder descargarlos hay que buscar fuentes confiables ya que pueden venir con virus pero en este server está totalmente libre de ellos
    Ya para poder descargarlos es simple, solo hay que activar la opción de fuentes desconocidas y listo
    Ya puedes disfrutar de tu app

    Datos OBB

    Para descargar juegos que incluyan archivos OBB siempre hay que extraer el archivo .zip o .rar en el caso en el que esté comprimido.
    Luego nos va a dar 2 archivos .apk y una carpeta de nombre inicial com. el APK lo descargamos y la carpeta com. lo movemos a la siguiente
    carpeta con algún explorador de archivos

    Android>obb

    Descargar juegos APKS

    para descargar archivos .apks es necesario descargar la aplicación de SAI de la play store para poder instalarlos.
    Luego es tan solo entrar a la aplicación y seleccionar donde dice instalar APKs y seleccionar el archivo

    Descargar juegos XAPK

    para descargar archivos .xapk es tan solo entrar a un explorador de archivos como tal y renombrar el .xapk a un .zip o a un .rar y
    descomprimir el archivo, ciertamente da igual si es .rar o .zip con cualquiera funciona.
    después de descomprimir el archivo le dejara los datos y el archivo apk

    Dispositivos Rooteados

    Cuando nosotros roteamos nuestro dispositivo lo que estamos haciendo es dejarlo como código en código abierto y dejando que cualquier aplicación
    o usuario modifique el código de este, aquí hay que tener cuidado ya que el mas mínimo virus puede dejar nuestro dispositivo como un pisa papeles
    ya que los troyanos o malwars modifican el código de este y dejándolo inútil, por eso se sugiere un gran cuidado si tienes tu dispositivo ruteado.

    Aclaración: Ningún dispositivo viene roteado de fabrica, esto se obtiene a base de modificar los archivos del sistema a través de una pc con tales programas
